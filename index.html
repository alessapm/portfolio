Remote Reads | A Digital Bookclub

A Node.js application designed for book lovers to create private groups and forums for book discussions.

Visit Website

View on GitHub

▬ Technologies Used ▬

  • Node.js
  • Express.js
  • JavaScript
  • PSQL
  • HTML/CSS

Remote Read is designed to function as a book club for friends who live too far apart to get together on a regular basis. "With Remote Read, you never need to let distance ruin your conversation, or your shared love of books!"

This is a fullstack MVC application that runs using Node.js with Express.js and renders views server side via ejs. Users can create accounts, create and join groups, post book club selections to their groups, and comment on the existing selections in the group. My favorite part of building out this project was figuring out how all of the data would relate to each other, so that groups could have multiple posts, groups could have mulitple members, and members could join mulitple groups!

Chewser

A React app that searches your area for local restaurants and selects an option for you.
No more wasting time deciding on where to eat- let Chewser choose for you!

Visit Website

View on GitHub

▬ Technologies Used ▬

  • React
  • Yelp API
  • JavaScript
  • Node.js
  • HTML/CSS

This application was created for anyone who feels inundated with all of today's choices.

It uses the Yelp API to grab places based on your geolocation and suggest a restaurant. The user can then choose to search for another place, if desired. Chewser was created in collaboration with Marcel Hamel. The best part of building Chewser was the satisfaction of making an elegant single-page app using React. The next step for Chewser will be developing it using React Native.

getWanderLust

A photo-inspired travel site that allows users to search beautiful, high-res photos of a city to get inspiration
and view these alongside a list of the city’s popular tourist attractions.

Visit Website

View on GitHub

▬ Technologies Used ▬

  • Node.js
  • React
  • JavaScript
  • Getty Images API
  • Yelp API
  • Axios
  • PSQL
  • HTML/CSS

The app uses React and Node.js as it's main technologies. It combines Getty Images’ and Yelp’s APIs in order to bring about a paired search that will inspire both the amateur and experienced traveler. getWanderlust is the a truly photo-inspired travel site, allowing you to feed your wanderlust by viewing crave-worthy photos of your desired destinations.
