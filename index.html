Remote Read

A Digital Bookclub

Visit Website

View on GitHub

▬ Technologies Used ▬

  • Node.js
  • Express.js
  • JavaScript
  • PSQL
  • HTML/CSS

A Node.js application designed for book lovers to create private groups and for book club style discussions.

Remote Read is designed to function as a book club for friends who live too far apart to get together on a regular basis. With Remote Read, you never need to let distance ruin your conversation, or your shared love of books!

Users can create accounts, create and join groups, post book club selections to their groups, and comment on the existing selections in the group.

Chewser

We Choose, You Eat

Visit Website

View on GitHub

▬ Technologies Used ▬

  • React
  • Yelp API
  • JavaScript
  • Node.js
  • HTML/CSS

A helpful app that searches your area for local restaurants and selects a single option for you. No more wasting time deciding on where to eat- let Chewser choose for you!

This React application uses the Yelp API to grab places based on geolocation and suggest a restaurant. The user can then choose to search for another place, if desired. Chewser was created in collaboration with Marcel Hamel.
